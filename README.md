#  Patterns and Sequence Builder

A beginner-friendly Python program that generates different number and star patterns using loops and iteration techniques. Perfect for understanding nested loops and pattern programming!


##  Features

### Core Features
-  **Star Triangle Pattern** - Right-aligned triangle using stars
-  **Reverse Star Pattern** - Inverted triangle using stars
-  **Number Triangle Pattern** - Sequential numbers in triangular form
-  **Number Pyramid Pattern** - Centered pyramid with numbers
-  **Print All Patterns** - Generate all patterns at once
-  **Input Validation** - Ensures only positive numbers are accepted
-  **Multiple Pattern Generation** - Generate unlimited patterns without restarting

### Technical Features
- Uses `for` loops and nested loops
- Functions for modular code organization
- Clear console output with separators
- Error handling for invalid inputs
